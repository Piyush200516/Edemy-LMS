# 🎓 LMS - Learning Management System

A simple and modern Learning Management System (LMS) built for managing and accessing online courses easily.

---

## 🚀 Features

✨ User Authentication (Login / Signup)  
📚 Add & View Courses  
📊 Simple Dashboard  
📱 Responsive UI  

---

## 🛠️ Tech Stack

**Frontend:** React.js  
**Backend:** Node.js, Express.js  
**Database:** MongoDB   

---

## 📂 Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/Piyush200516/Edemy-LMS
cd cd edemy-lms

Install Dependencies

npm install

3. Run the Project

npm start
